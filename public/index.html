Hello Fresco!

This web application is an API-REST made with Node, Express and MongoDB (MongoAtlas) for educational purposes.

Try the different routes to get JSON information using GET request:

You can use the URL

GET "CONGELADOS"

Read all "CONGELADOS": api/congelados

Read a "CONGELADO" by id: api/congelados/{id} (Langostino's Example)

GET "VEHICULOS"

Read all "VEHICULOS": api/vehiculos

Read a "VEHICULO" by id: api/vehiculos/{id} (MAN TGE's Example)

GET "REPARTIDORES"

Read all "REPARTIDORES": api/repartidores

Read a "REPARTIDOR" by id: api/repartidores/{id} (Author Example)

You can Create, Update and Delete a "CONGELADO", "VEHICULO" or "REPARTIDOR" with this routes using POST, PUT and DELETE requests:

You will need postman, curl or similar app

POST PUT & DELETE "CONGELADOS"

Create a "CONGELADO": api/congelados (needs POSTS requets)

Update a "CONGELADO": api/congelados/{id} (needs PUT requets)

Delete a "CONGELADO": api/congelados/{id} (needs DELETE requets)

POST PUT & DELETE "VEHICULOS"

Create a "VEHICULO": api/vehiculos (needs POSTS requets)

Update a "VEHICULO": api/vehiculos/{id} (needs PUT requets)

Delete a "VEHICULO": api/vehiculos/{id} (needs DELETE requets)

POST PUT & DELETE "REPARTIDOR"

Create a "REPARTIDOR": api/repartidores (needs POSTS requets)

Update a "REPARTIDOR": api/repartidores/{id} (needs PUT requets)

Delete a "REPARTIDOR": api/repartidores/{id} (needs DELETE requets)
